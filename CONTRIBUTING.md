# Contributing to SHS Player

First off, thank you for considering contributing to SHS Player! It's people like you that make this tool such a great multimedia engine.

## How Can I Contribute?

### 🐛 Reporting Bugs
* Ensure the bug was not already reported by searching on GitHub under Issues.
* If you're unable to find an open issue addressing the problem, open a new one. 
* Be sure to include a title and clear description, as much relevant information as possible, and a code sample or an executable test case demonstrating the expected behavior that is not occurring.

### ✨ Suggesting Enhancements
* Open a new issue with a clear title and a detailed description of the suggested enhancement.
* Explain why this enhancement would be useful to most users.

### 💻 Pull Requests
* Fork the repo and create your branch from `main`.
* If you've added code that should be tested, add tests.
* Ensure the test suite passes.
* Make sure your code lints (run `ktlintCheck`).
* Issue that pull request!

## Code of Conduct
By participating in this project, you agree to abide by our Code of Conduct.
